# FocusFlow

FocusFlow is a warm, modern productivity web app prototype for students and developers.

## Current implementation

- React 19 + Vite frontend
- Tailwind CSS 4 styling with amber/warm theme
- Sidebar + top navigation app shell
- Dashboard with greeting, streak, XP, productivity, tasks, upcoming timer, achievements
- Planner, Calendar, Habits, Analytics, Challenges, Friends, Notes, Achievements, and Settings pages
- Weekly/monthly charts and focus breakdown using Recharts

## Run locally

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run build
```
